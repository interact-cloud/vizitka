## Дизайн

Актуальный стандарт — **фирменный стиль Siberian Wellness** (как на флаерах 30-го бизнес-года: Club 200 / Club 1000). Светлый сиреневый фон, фиолетовый и бирюзовый акценты, белые карточки, кремово-золотой логотип, плотный гротеск.

**Цвета** (переменные из `:root`):
```css
--bg:        #c9c2ea   /* светлый сиреневый фон страницы */
--banner:    #4f46a8   /* насыщенный фиолетовый — шапки, баннеры */
--card:      #ffffff   /* белые карточки */
--card-soft: #f3effb   /* мягкая лавандовая карточка */
--violet:      #6c5ce7 /* фиолетовый акцент — Club 1000 */
--violet-deep: #5b21b6 /* тёмный фиолетовый — цифры, заголовки */
--teal:        #1fb8ae /* бирюзовый акцент — Club 200, выгода */
--gold:        #e6c089 /* кремово-золотой — логотип "Club" */
--gold-btn:    #e7b063 /* золотисто-персиковые кнопки CTA */
--ink:    #2d2a5a      /* основной тёмный текст на светлом */
--muted:  #6b6790      /* второстепенный текст */
--white:  #ffffff
```

**Шрифты** (подключены через Google Fonts):
- `Montserrat` — весь текст. Заголовки в плотных начертаниях **700/800** (жирный гротеск как на флаере), основной текст 400/500.

**Декор:**
- шестилучевая снежинка ❄ (фирменный знак SW)
- скруглённые карточки (radius 16–24px), мягкие тени
- крупные цифры шагов (1, 2, 3…) в золоте или акцентном цвете
- плашка «30-й бизнес-год» — тёмная с золотой обводкой

**Анимации** — умеренные: появление блоков (`fadeUp`, `cardIn`), мягкое свечение на кнопках.

**Не использовать** на новых страницах:
- тёмные фоны (это легаси старых страниц — см. ниже)
- другие шрифты вне Montserrat
- резкие цвета вне фирменной палитры

> 🗄️ **Легаси:** ранее проект был в тёмной премиум-теме с золотом (`--bg:#080b12`, `--surface:#0e1220`, `--gold:#c9a84c`, шрифт Cormorant Garamond). Это устаревший стиль — если где-то встретятся старые страницы в нём, не переверстываем без задачи. Новые страницы — только в фирменном стиле SW выше.

## Стиль текста на страницах

Простой и живой язык — как будто объясняешь другу.

**Хорошо:**
> «Хочешь попробовать продукт? Напиши мне — разберёмся вместе»

**Плохо:**
> «Для получения консультации по вопросам продукции необходимо обратиться к представителю»

Правила:
- короткие предложения
- никаких «осуществляется», «в рамках», «в целях»
- обращение на «ты» (если к клиенту) или «вы» — решить до написания
- конкретика вместо общих слов

## Структура новых страниц

Каждая HTML-страница начинается с этого шаблона:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>...</title>
  <style>
    @import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700;800&display=swap');
    *, *::before, *::after { box-sizing: border-box; margin: 0; padding: 0; }
    :root {
      --bg: #c9c2ea; --banner: #4f46a8;
      --card: #ffffff; --card-soft: #f3effb;
      --violet: #6c5ce7; --violet-deep: #5b21b6; --teal: #1fb8ae;
      --gold: #e6c089; --gold-btn: #e7b063;
      --ink: #2d2a5a; --muted: #6b6790; --white: #ffffff;
    }
    html, body {
      background: var(--bg);
      color: var(--ink);
      font-family: 'Montserrat', sans-serif;
      font-weight: 400;
    }
  </style>
</head>
<body>
  <!-- контент -->
</body>
</html>
```
