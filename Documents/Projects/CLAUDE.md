# Правила работы с проектом

## Перед созданием любого нового файла

Сначала рассказываю что планирую сделать — какой файл, зачем, что внутри.
Только после подтверждения создаю.

## Дизайн

Все страницы выдержаны в одном стиле — тёмный премиум с золотом.

**Цвета** (переменные из `:root`):
```css
--bg:      #080b12   /* фон страницы */
--surface: #0e1220   /* фон карточек и блоков */
--gold:    #c9a84c   /* основной акцент */
--gold2:   #f0d080   /* светлое золото, градиенты */
--teal:    #38c8c0   /* второй акцент */
--text:    #e8e4dc   /* основной текст */
--muted:   #7a8090   /* второстепенный текст */
--border:  rgba(201,168,76,0.18)  /* рамки */
```

**Шрифты** (подключены через Google Fonts):
- `Cormorant Garamond` — заголовки, имена, акцентные надписи
- `Montserrat` — весь остальной текст

**Анимации** — есть, умеренные:
- появление блоков (`fadeUp`, `cardIn`)
- пульсация, свечение, shimmer на кнопках
- дышащий фон (`breathe`)

**Не использовать** на новых страницах:
- светлые фоны
- другие шрифты
- резкие яркие цвета вне палитры

## Стиль текста на страницах

Простой и живой язык — как будто объясняешь другу.

**Хорошо:**
> «Хочешь попробовать продукт? Напиши мне — разберёмся вместе»

**Плохо:**
> «Для получения консультации по вопросам продукции необходимо обратиться к представителю»

Правила:
- короткие предложения
- никаких «осуществляется», «в рамках», «в целях»
- обращение на «ты» (если к клиенту) или «вы» — решить до написания
- конкретика вместо общих слов

## Структура новых страниц

Каждая HTML-страница начинается с этого шаблона:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>...</title>
  <style>
    @import url('https://fonts.googleapis.com/css2?family=Cormorant+Garamond:wght@300;400;600;700&family=Montserrat:wght@300;400;500&display=swap');
    *, *::before, *::after { box-sizing: border-box; margin: 0; padding: 0; }
    :root {
      --bg: #080b12; --surface: #0e1220;
      --gold: #c9a84c; --gold2: #f0d080; --teal: #38c8c0;
      --text: #e8e4dc; --muted: #7a8090;
      --border: rgba(201,168,76,0.18);
    }
    html, body {
      background: var(--bg);
      color: var(--text);
      font-family: 'Montserrat', sans-serif;
      font-weight: 300;
    }
  </style>
</head>
<body>
  <!-- контент -->
</body>
</html>
```
