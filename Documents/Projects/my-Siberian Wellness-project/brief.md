## 1. Цель проекта и целевая аудитория

### Цель
Telegram-бот, который **автоматически проводит нового Бизнес-Партнёра Siberian Wellness через первый месяц** и далее сопровождает ежедневно: объясняет программы, ставит задачи, помогает с первым заказом, подбором продуктов и работой с клиентами. Бот снимает рутину с живого наставника и держит новичка «за руку» 24/7, доводя до первого результата.

**Бизнес-цель:** повысить долю новичков, которые делают первый заказ и доходят до Club 200, и снизить ранний отток (главная боль рынка по [research.md, раздел 1](sw_research.md)).

### Целевая аудитория
Люди, **уже пришедшие в бизнес** Siberian Wellness как Бизнес-Партнёры. Берут продукты для себя и сразу для знакомых, готовы приглашать клиентов с первых дней. Рекрутинг включается сразу. Обращение — на «ты».

### Роли (две, не путать)
- **Наставник 24/7** — бот, автопомощник, на связи круглосуточно.
- **Евгения Громова, [@gromovaevgenya](https://t.me/gromovaevgenya)** — живой наставник. В текстах человека называем по имени «Евгения», не «наставник».

## 3. Полная структура бота (экраны по порядку)

> Единая нумерация дней: «День N» = N-й день от регистрации (День 0 = регистрация). Один счётчик для онбординга и ежедневного цикла.

### Блок A. Глобальное поведение (работает в любой точке)
- Команды `/menu`, `/start`, `/help` — доступны всегда.
- Кнопка «🏠 Главное меню» — в каждом сообщении.
- Фолбэк на свободный текст: распознавание по ключевым словам → нужный экран; не распознал → мягкий ответ + меню.
- Повторный `/start` для вернувшегося → «С возвращением» + меню (не повторять онбординг).
- Время рассылок — по часовому поясу партнёра.

### Блок B. Онбординг по дням (поток)
```
День 0  → 1.1 Приветствие → 1.2 Club 200 → 1.2б Шаг 1: первый заказ
День 1  → 2.1 Club 1000 (снежинки, сертификаты)
День 3  → 3.1 Привилегированные клиенты (ПК)
День 5  → 3.3 Клуб Постоянства
День 7  → 4.1 Чек-лист первой недели
Неделя 2–4 → 5.1 / 5.2 / 5.3 (снежинки, двойное напоминание, итоги месяца)
```

### Блок C. Ежедневный цикл (каждый день, параллельно онбордингу)
```
УТРО (9:00)  → афоризм дня + продукт дня + 3 задачи
ВЕЧЕР (20:00) → запрос отчёта «что сделал»
Каждые 3 дня → напоминание связаться с Евгенией
```

### Блок D. Триггерные сообщения (по событиям)
Т-0 первый заказ (быстрый win) · Т-1 новый ПК · Т-2 50% до сертификата · Т-3 новый ранг · Т-4 дедлайн 200 баллов · Т-5 оборот 500–1000.

### Блок E. Главное меню (разделы)
☀️ Задачи на сегодня · 📊 Мой прогресс и ранг · 🎯 Программы · 🎁 Клуб Постоянства · ❄️ Мои снежинки · 🛍️ Подобрать продукт · 🛡️ Возражения · 🗣️ Как рекомендовать · 📚 Обучение · 📖 Объясни простыми словами · 💬 Написать Евгении · ❓ FAQ.

### Блок F. Служебные экраны
Экраны меню с плейсхолдерами (если данных ещё нет), словарь терминов, FAQ.

## 5. Стиль дизайна (для визуальных карточек и оформления)

Сообщения бота — текст + эмодзи + кнопки. Где нужна картинка (приветствие, шаги программ, шкалы) — визуальные карточки 9:16 в фирменном стиле SW. Готовые карточки — в [sw_cards.html](sw_cards.html).

### Цвета (фирменный стиль Siberian Wellness, 30-й бизнес-год)
| Переменная | HEX | Роль |
|---|---|---|
| `--bg` | `#c9c2ea` | светлый сиреневый фон |
| `--banner` | `#4f46a8` | насыщенный фиолетовый — шапки |
| `--card` | `#ffffff` | белые карточки |
| `--violet` | `#6c5ce7` | акцент Club 1000 |
| `--teal` | `#1fb8ae` | акцент Club 200, выгода |
| `--gold` | `#e6c089` | золото — достижения, логотип |
| `--ink` | `#2d2a5a` | основной текст |

**Цветовая логика по программам:** Club 200 → бирюза · Club 1000 → фиолет · Клуб Постоянства → оранж · достижения/поздравления → золото + снежинка ❄.

### Шрифт
**Montserrat** — весь текст. Заголовки 700/800, основной 400/500.

### Настроение
Светлое, дружелюбное, мотивирующее, «человек рядом». Не тяжёлый бэк-офис (антипример — глючные фирменные приложения из research), а лёгкий живой наставник. Эмодзи — умеренно, как акценты.

## 6. Обязательные элементы

### Кнопки (UX-правила из research, раздел 3)
- В **каждом** сообщении — кнопка «🏠 Главное меню». Ни одной тупиковой кнопки (каждая ведёт к следующему шагу).
- Поддержка и кнопок, и свободного текста (новичку — кнопки, опытному — быстрый ввод).
- Везде доступен видимый выход на живого человека — «💬 Спросить Евгению».
- Кнопки-ссылки: «🛒 Открыть магазин» → https://ru.siberianhealth.com/ru/

### Формы / сбор данных
- Имя — из Telegram (`{имя}`).
- Часовой пояс — из Telegram или уточнить при регистрации.
- Прогресс ({личный_объём}, снежинки, ПК, ранг) — из выгрузки ЛК/CRM (см. раздел 7).
- При сборе контактов на будущих этапах — экран согласия (отдельно).

### Анимации / визуал
- В визуальных карточках (sw_cards.html): умеренные `fadeUp`/`cardIn`, мягкое свечение кнопок, снежинка ❄ как фирменный знак.
- В самом боте — «печатает…» (typing) перед ответом, короткая задержка перед проверкой действия (UX: иначе бот «роняет» сценарий).

### Обязательная логика (из research + ревью)
- Честные плейсхолдеры, когда данных ещё нет («цифры появятся после первого заказа»).
- Прогресс показывать визуально (шкала «сколько до уровня»), а не голыми цифрами.
- Проактивные подсказки — только когда они по делу (награда близко / задача актуальна), не спам.

## 7. Технологии (рекомендованный стек)

По [research.md, раздел 4](sw_research.md). **Стек под старт** (один предприниматель, без команды, минимальный бюджет):

| Категория | Выбор | Ориент. цена (нач. 2026) |
|---|---|---|
| Конструктор | **BotHelp** (или Salebot, если нужны курсы-онбординг) | от 1 599 ₽/мес |
| База данных | встроенная в конструктор + Google Sheets | бесплатно |
| Платежи | **ЮKassa** (СБП от 0,4%) | без абонплаты |
| Рассылки | встроенные в конструктор (лимиты Telegram учтены) | в тарифе |
| Хостинг | на стороне платформы | в тарифе |

**Итого ~1 600–3 000 ₽/мес.** Переход на свой код (aiogram + PostgreSQL + VPS) — только при упоре в лимиты конструктора и наличии разработчика.

> ⚠️ **Синхронизация данных:** триггеры реального времени (Т-0 первый заказ, дедлайны Т-4/Т-5) корректны только при ежедневной выгрузке из ЛК. Если выгрузка реже — помечать такие сообщения как приблизительные или отключать. Зафиксировать: кто грузит и как часто.

## 8. Что отстраивает нас от конкурентов

По [research.md, раздел 1](sw_research.md):
- **Лёгкость и человечность** против тяжёлых глючных фирменных приложений (Oriflame, NL).
- **Маркетинг-план простым языком** — закрываем главную боль рынка («сложно, таблицы и проценты»).
- **Пошаговый онбординг с первого контакта** + ежедневные задачи (как у Epixel, но по-русски и по-человечески).
- **Живой наставник в связке с ботом** — бот не отфутболивает, а ведёт; Евгения подключается, когда нужен живой разбор.
