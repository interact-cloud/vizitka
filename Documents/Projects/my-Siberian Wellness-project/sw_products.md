```
📦 Название: …
Категория: … (из списка выше)
Баллы: … (из sw_pricelist_raw.md)      Цена для ПК: … ₽
Кому подойдёт: … (1 фраза — под какую цель)
В чём польза: … (2–3 пункта простым языком, без обещаний лечения)
Как принимать: …
Фраза для рекомендации: «…» (готовый текст, чтобы партнёр скопировал клиенту)
```

**Пример:**
```
📦 Название: Men's Box 3 в 1
Категория: Энергия / мужское здоровье
Баллы: 59          Цена для ПК: 2690 ₽
Кому подойдёт: мужчине, который к вечеру выжат и хочет больше энергии и тонуса
В чём польза:
  • поддержка мужской силы и тонуса
  • удобно — один пакетик в день, ничего не забудешь
  • природные компоненты + аминокомплекс
Как принимать: 1 пакетик в день
Фраза для рекомендации: «Если день длинный, а сил к вечеру не остаётся —
  попробуй, мне реально помогает держать темп»
```
